## Installation

1. Download Nodejs in https://nodejs.org/en/

OR

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
```

```bash
nvm install --lts
```

2. Download Visual Studio Code
